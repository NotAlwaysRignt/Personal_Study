postgresql 没有 auto_increament ,用 SERIAL 代替
```sql
CREATE TABLE tablename (
   colname SERIAL
);
```

postgresql 在 insert 时, CHAR 或 TEXT 类型的值用单引号而不是双引号
```sql
INSERT INTO COMPANY (NAME,AGE,ADDRESS,SALARY)
VALUES ( 'Kim', 22, 'South-Hall', 45000.00 );
```
